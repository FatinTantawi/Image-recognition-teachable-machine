# Image Recognition using Teachable Machine

## Overview

This project demonstrates an image recognition model created using Google Teachable Machine. The model was trained to classify images of **Albert Einstein** and **Pablo Picasso**. After training, the model was exported in TensorFlow/Keras format and tested using Python in Google Colab to predict the class of input images.

## Classes

- Albert Einstein
- Pablo Picasso

## Tools Used

- Google Teachable Machine
- TensorFlow / Keras
- Python
- Google Colab

## Project Steps

1. Created an Image Classification project using Google Teachable Machine.
2. Added two classes: **Albert Einstein** and **Pablo Picasso**.
3. Collected and uploaded training images for each class.
4. Trained and evaluated the model.
5. Exported the trained model in TensorFlow/Keras format.
6. Uploaded the model files to Google Colab.
7. Wrote a Python script to load the model and classify input images.
8. Tested the model and displayed the predicted class with its confidence score.
  
## Output

The model successfully identifies whether the input image belongs to **Albert Einstein** or **Pablo Picasso** and displays the prediction with its confidence score.
